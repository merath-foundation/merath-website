# Merath Cultural Foundation Website

Website for Merath Cultural Foundation. Built with React, TypeScript, and Vite.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18.3.1
- TypeScript
- Vite 6.3.5
- React Router
- Tailwind CSS

## Features

- English/Arabic support with RTL
- Projects gallery
- Archive section
- Contact forms
